Each Computer has a built-in instruction set that it knows how to execute by design.



Recall that we use the term computation to mean "the execution of instruction by a computer with the intentioin of solving a given problem."

Incorrect. Each computer has a set of primitive instructions that it is "wired" to execute. We call this the Instruction       set of the computer.



The Computer uses intelligence to execute instructions.



Recall that we said a computer executes the instructions given to it immediately.

    Correct! A Computer executes the instructions automatically. Its main task is to simply execute the instructions     given to it (from its instruction set) and remember their results as the computation goes on. The part that may requires intelligence is expressing in terms of these instructions.
